Bem-vindo à documentação da API OpsLinux

A OpsLinuxAPI é uma API projetada para monitorar e obter informações detalhadas sobre o sistema Linux. Ela oferece dados em tempo real sobre o uso de CPU, memória, discos, interfaces de rede, processos em execução e tempo de atividade do sistema.

Construída com:

Python Flask

No menu ao lado, você encontrará links para descrições detalhadas de cada módulo da API, permitindo explorar todas as funcionalidades disponíveis de maneira fácil e organizada.
